# C-Programs

This repository contains basic C programs created for practice and learning.
It is especially useful for beginners who are starting with C programming.

## Programs Included
- Hello World program (printing a greeting message)

## File Details
- `Hello.c` : A simple C program to print a greeting message on the screen.

## How to Run the Program
1. Install a C compiler (GCC / Turbo C / Dev-C++)
2. Open terminal or command prompt
3. Compile the program using:
   gcc Hello.c -o Hello
4. Run the program:
   ./Hello

## Purpose of This Repository
- To practice basic C programming concepts
- To build a strong foundation in C language
- To maintain a collection of simple C programs for revision
